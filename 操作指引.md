# 云纪念 APK 打包操作指引

## 第一步：在 GitHub 创建仓库并获取 Token

### 1.1 创建 GitHub 账号（已有则跳过）
打开 https://github.com，注册账号。

### 1.2 创建新仓库
1. 登录 GitHub → 点右上角 **+** → **New repository**
2. Repository name 填：`yun-memorial-android`
3. 选 **Public** 或 **Private** 均可
4. 不要勾选任何初始化选项（README/gitignore/license 都不选）
5. 点 **Create repository**

### 1.3 获取 Personal Access Token
1. 右上角头像 → **Settings** → 左侧最底部 **Developer settings**
2. **Personal access tokens** → **Tokens (classic)** → **Generate new token (classic)**
3. Note 填：`android-build`
4. 勾选：**repo**（全选）
5. 点 **Generate token**，**复制 token**（只显示一次！）

---

## 第二步：Push 源码到 GitHub

在以下路径打开 Git Bash 或 PowerShell：
```
E:\4、qita\apk\yun-memorial-android\
```

依次执行以下命令（把 `YOUR_USERNAME` 和 `YOUR_TOKEN` 替换成实际值）：

```bash
git init
git add .
git commit -m "初始化云纪念 Android 项目"
git branch -M main
git remote add origin https://YOUR_TOKEN@github.com/YOUR_USERNAME/yun-memorial-android.git
git push -u origin main
```

---

## 第三步：等待 GitHub Actions 自动打包

1. 打开 GitHub 仓库页面
2. 点顶部 **Actions** 标签
3. 看到 **Build APK** 工作流正在运行（黄色圆圈）
4. 等待 3~5 分钟，变成绿色 ✅ 表示成功

---

## 第四步：下载 APK

1. 点进刚才的工作流记录
2. 页面底部 **Artifacts** 区域
3. 下载 `yun-memorial-release`（正式版）或 `yun-memorial-debug`（调试版）
4. 解压得到 `.apk` 文件

---

## 第五步：安装到安卓手机

1. 将 `.apk` 文件传到手机（微信/数据线/QQ 均可）
2. 手机上打开文件管理器找到 apk
3. 点击安装（需允许"安装未知来源应用"）
4. 安装完成，桌面会出现"云纪念" App 图标

---

## 常见问题

**Q: Actions 报错 FAILURE: Build failed**
A: 多半是 Gradle 版本兼容问题，可在 Actions 日志里查看具体报错，告知喵来修复。

**Q: APK 安装后打不开页面**
A: 确认手机能访问 http://47.96.125.42（需要连同一网络，或服务器需有公网访问权限）。

**Q: 以后想换服务器地址怎么办**
A: 修改 `app/src/main/java/com/yunmemorial/app/MainActivity.kt` 第 20 行的 `BASE_URL`，重新 push 即可自动打包。
